Detailed Description

The interface for the SCHEME FFT initialization subroutine.

The CREATE_SCHEME_FFT subroutine initializes a SCHEME for future FFTs. For greater detail on the SCHEME FFT initialization subroutine, see create_scheme_fft.f90. For greater detail on the SCHEME datatype, see ez_parallel_structs.f90.
