Detailed Description

The interface for the spectral derivative array creation subroutine.

The CREATE_SPEC_DERV subroutine fills in an array for future spectral derivatives. For greater detail on the spectral derivative array creation subroutine, see create_spec_drv.f90. For greater detail on the SCHEME datatype, see ez_parallel_structs.f90.
